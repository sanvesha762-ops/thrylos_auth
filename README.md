# THRYLOS Auth - Document Verification System

A secure, modern document verification platform built with vanilla HTML, CSS, and JavaScript.

## Features

- **QR Code Scanning** — Instantly scan and verify documents via QR code
- **File Upload** — Drag-and-drop or browse to upload documents for verification
- **Manual Verification** — Enter a verification ID to check document authenticity
- **Dark Mode UI** — Premium glassmorphism design with Inter typography
- **Responsive Design** — Fully responsive across all device sizes
- **Zero Dependencies** — Pure HTML/CSS/JS, no build tools required

## Tech Stack

- HTML5
- CSS3 (Custom Properties, Flexbox, Grid, Animations)
- Vanilla JavaScript (ES6+)
- [Inter](https://fonts.google.com/specimen/Inter) font
- [Lucide Icons](https://lucide.dev)

## Getting Started

```bash
# Clone the repository
git clone https://github.com/sanvesha762-ops/thrylos_auth.git
cd thrylos_auth

# Open in browser
open index.html
# Or use a local server
python3 -m http.server 8080
```
